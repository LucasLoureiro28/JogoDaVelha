# 🎮 Jogo da Velha em Java

Projeto de Jogo da Velha desenvolvido em Java, com duas formas de jogo:
- Pelo terminal
- Interface gráfica usando Swing

O projeto foi desenvolvido com o objetivo de praticar lógica de programação, orientação a objetos e criação de interfaces gráficas em Java.

---

## 📌 Funcionalidades

- Jogo para dois jogadores
- Tabuleiro 3x3
- Verificação automática de vitória
- Verificação de empate (velha)
- Sistema de revanche
- Interface gráfica com botões
- Mudança de cor conforme o jogador
- Bloqueio do tabuleiro ao finalizar o jogo

---

## 🛠 Tecnologias utilizadas

- Java
- Swing (Interface gráfica)
- Orientação a Objetos

---

## ▶️ Como executar o projeto

### Interface gráfica
1. Abrir o projeto em uma IDE (IntelliJ, Eclipse ou VS Code)
2. Executar a classe `Main.java`
3. O jogo abrirá em uma janela

### Pelo terminal
1. Executar o método `Jogar()` da classe `Jogo`
2. Informar linha e coluna para jogar

Exemplo de jogada:
0 0
1 1
2 2
